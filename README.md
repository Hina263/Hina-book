# Hina-book

## 概要
授業課題として制作したDjango製のWebアプリです。
個人の本棚として、本の情報を登録・閲覧・管理できるアプリケーションです。
各本にレビューと評価（0〜5）を付けられ、検索やソートで探しやすい構成になっています。

## 使用技術
- Python
- Django 5.1
- HTML / CSS
- Bootstrap 5.3
- SQLite

## 機能
- **本の登録・一覧・詳細表示**
- **本の更新・削除**（登録者本人のみ）
- **レビュー機能**（タイトル・本文・0〜5の評価）
- **検索・ソート**（キーワード検索、タイトル順・評価順・日付順で並び替え）
- **会員登録・ログイン・ログアウト**
- **評価順TOP2**（サイドバーで人気の本を表示）
- **ページネーション**

## プロジェクト構成
```
Hina-book/
├── bookproject/     # プロジェクト設定
├── book/            # 本・レビュー機能
├── accounts/        # 会員登録・認証
├── templates/       # 共通テンプレート（base.html など）
└── media/           # 本のサムネイル画像
```

## セットアップ

```bash
# 依存関係のインストール
pip install django pillow

# マイグレーション
python manage.py migrate

# 開発サーバー起動
python manage.py runserver
```

## 制作背景
